## Опис

У лабораторній роботі реалізовано алгоритмічне завдання — логіку падіння однієї фігури Tetris, яка опускається вниз, доки не зіткнеться з ландшафтом або нижньою межею поля.

У вхідному файлі міститься:

- розмір поля (`height width`);
- сітка, де:
  - `.` — порожній піксель,
  - `p` — піксель “підвішеної” фігури,
  - `#` — піксель ландшафту.

Після обробки програма виводить фінальний стан поля, коли фігура “впала”.

Особливості реалізації:

- розділено логічний шар, шар введення/виведення та комунікаційний шар;
- реалізовано Dependency Injection для можливості підміни I/O у тестах;
- створені власні test doubles (без сторонніх бібліотек);
- усі шари покрито юніт-тестами;
- налаштовано GitHub Actions для автоматичного запуску тестів;
- створено PR з успішним прогоном тестів та PR із тестами, що падають.

## Вимоги

* Python 3.11+
* `pip`
* `pytest`

## Налаштування середовища / встановлення залежностей

```bash
pip install -r requirements.txt
```

## Запуск тестів

```bash
python -m pytest
```

## Запуск програми вручну

Вхідний файл `input.txt` (приклад):

```bash
7 8
..p.....
.ppp....
..p.....
........
...#....
...#...#
#..#####
```

### Windows

```bash
python tetis.py input.txt
```
або

```bash
./tetis input.txt
```

### Linux

```bash
python3 tetis.py input.txt
```
або

```bash
./tetis input.txt
```
